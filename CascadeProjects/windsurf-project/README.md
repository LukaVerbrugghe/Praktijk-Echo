# Piano Studio Website

A professional, responsive website for a piano teacher built with HTML, CSS, and JavaScript. Designed to be hosted on GitHub Pages.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with light colors and music theme
- **Interactive Elements**: Smooth animations, hover effects, and transitions
- **Background Video**: Repeating piano video background on the home page
- **Multiple Pages**: Home, Music Therapy, Music Lessons, and About pages
- **Contact Footer**: Consistent footer with contact information on all pages
- **Accessibility**: Semantic HTML5 structure with proper ARIA labels
- **Performance**: Optimized images, lazy loading, and efficient animations

## Pages

### Home (`index.html`)
- Hero section with animated title and subtitle
- Background video of piano playing
- Teacher introduction section with image and credentials

- Introduction to music therapy benefits
- Benefits grid with icons and descriptions

## Technologies Used

- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Modern styling with animations and transitions
- **JavaScript**: Interactive features and smooth scrolling
- **Google Fonts**: Playfair Display and Open Sans for typography
- **Font Awesome**: Icons for visual enhancement

## Customization

### Images and Videos
Replace placeholder images and videos with your actual content:
- Teacher photos: Replace `https://via.placeholder.com/...` URLs
- Background video: Update the video source in `index.html`
- Studio images: Add your own studio photos

### Colors
The color scheme can be easily modified in `styles.css`:
- Primary color: `#8e44ad` (purple)
- Secondary color: `#3498db` (blue)
- Text colors: `#2c3e50`, `#555`, `#666`
- Background colors: `#fafafa`, `#f8f9fa`, `white`

### Contact Information
Update contact details in the footer section of each HTML file:
- Phone number
- Email address
- Physical address
- Social media links

## Deployment to GitHub Pages

1. Push all files to your GitHub repository
2. Go to repository Settings
3. Scroll down to "GitHub Pages" section
4. Under "Source", select "Deploy from a branch"
5. Choose "main" branch and "/ (root)" folder
6. Click "Save"
7. Your site will be available at `https://username.github.io/repository-name`
